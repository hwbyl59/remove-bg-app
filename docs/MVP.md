# 🖼️ Image Background Remover — MVP 文档

> 版本：v1.0 | 日期：2026-03-25 | 状态：🟡 开发中

---

## 1. 产品概述

### 1.1 产品定位

**一句话描述：** 一款极简的 AI 驱动的图像背景移除工具，用户上传图片，即刻获得透明背景结果。

**核心差异点：**
- 🚀 **极简** — 无需注册，打开即用，无门槛
- 🔒 **隐私优先** — 图片即传即处理，不存储
- ⚡ **快速** — 全流程 5-10 秒完成

### 1.2 目标用户

| 用户群体 | 使用场景 |
|---------|---------|
| 电商卖家 | 制作白底产品图、主图抠图 |
| 设计师 | 快速提取素材、合成海报 |
| 普通人 | 证件照换底色、创意照片 |
| 社交媒体运营 | 制作透明背景图标、贴纸 |

### 1.3 核心指标（KPI）

| 指标 | 目标 |
|------|------|
| 单次使用转化率 | >70%（上传→处理完成） |
| 平均处理时长 | <10 秒 |
| 用户满意度 | >4.0/5.0 |
| API 成本回收 | 控制在收益 30% 以内 |

## 2. 功能范围

### 2.1 MVP 必做功能（Must Have）

#### 🎯 F1：图片上传
- 支持拖拽上传和点击选择
- 支持格式：JPG、PNG、WebP
- 文件大小限制：12MB
- 前端实时预览原图

#### 🎯 F2：AI 背景移除
- 调用 Remove.bg API 进行处理
- 输出 PNG（透明背景）
- 加载状态展示（spinner + 文字提示）
- 错误提示（文件过大、API 失败、网络错误）

#### 🎯 F3：结果预览 + 下载
- 棋盘格背景预览（透明效果可见）
- 一键下载 PNG 到本地

#### 🎯 F4：重置 / 继续使用
- 处理完成后可下载，也可重新上传
- 无需刷新页面

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Next.js 14 (App Router) | SSR + API Routes 一体化 |
| 样式 | Tailwind CSS | 原子化 CSS，快速构建 |
| 后端 | Next.js API Route | `/api/remove-bg` 端点 |
| AI 提供商 | Remove.bg API | 即用型背景移除 API |
| 部署平台 | Vercel | 前端+后端一键部署，免费额度 |

### 4.2 数据流

```
用户上传图片 (FormData POST)
    ↓
Next.js API Route (/api/remove-bg)
    ↓
验证 API Key & 文件
    ↓
转发到 Remove.bg API (X-Api-Key header)
    ↓
接收 PNG 结果 (Buffer)
    ↓
返回给前端 (image/png)
    ↓
前端展示预览 + 提供下载链接
```

### 4.3 API 设计

#### `POST /api/remove-bg`

**请求：**
```
Content-Type: multipart/form-data
Body: { image: File }
```

**成功响应：**
```
HTTP 200
Content-Type: image/png
Body: <PNG 二进制数据>
```

**错误响应：**
```json
HTTP 400/500
Content-Type: application/json
{
  "error": "错误描述文字"
}
```

### 4.4 安全设计

- ✅ API Key 仅存在于服务端（`.env.local`，不提交 Git）
- ✅ 前端不持有任何敏感凭证
- ✅ 文件大小限制 12MB（防止滥用）
- ✅ 不存储任何上传或处理后的图片
- ✅ 图片即传即处理，请求结束即丢弃

### 4.5 部署架构

```
                    ┌─────────────────┐
                    │   Vercel CDN    │
                    │  (全球加速)      │
                    └────────┬────────┘
                             │
              ┌──────────────┴──────────────┐
              │                             │
       ┌──────▼──────┐              ┌───────▼──────┐
       │  Next.js    │              │  Remove.bg   │
       │  前端页面    │              │  AI API      │
       │  /          │              │  (境外服务)  │
       ├─────────────┤              └──────────────┘
       │  API Route  │
       │  /api/      │
       │  remove-bg  │
       └─────────────┘
```

## 7. 风险与对策

| 风险 | 概率 | 影响 | 对策 |
|------|------|------|------|
| Remove.bg API 调不通（境外） | 中 | 高 | 备用方案：Clipdrop API / 自托管 RMBG-1.4 |
| 免费额度很快用完 | 高 | 中 | 早期设置用量提示，引导付费或切换方案 |
| 用户上传违规内容 | 低 | 高 | 前端加文件类型校验，后端不做存储 |
| Vercel 冷启动慢 | 低 | 低 | Upgrade 到 Pro 或换 Railway |
